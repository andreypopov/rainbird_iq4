Rain Bird IQ4 token helper

This helper avoids the fast Rain Bird redirect. Sign in normally, then use a bookmarklet on the Rain Bird page to send the token back to Home Assistant.

Setup

  1. Drag the bookmarklet below to your bookmarks bar. If your browser does not allow dragging, use Copy bookmarklet and create a bookmark manually.
  2. Open the Rain Bird login link and sign in. It is fine if Rain Bird redirects away from the blank auth page.
  3. When the Rain Bird web app has loaded, click the bookmarklet from your bookmarks bar.
  4. Return to the Home Assistant Rain Bird IQ4 dialog and click Submit. You can leave the token field empty.
